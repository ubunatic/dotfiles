# Folder for AI Reviews and AI Projects planning

This folder contains documentation, resources, results, and reports from
AI-assisted reviews, project planning, agentic coding.

## Structure
> [!Note]
> This is a suggested structure and can be adapted based on specific needs.

- `current/`: Projects and reviews that are currently active.
- `archive/`: Projects and reviews that have been completed or are no longer active.
- `instructions`: Code/path/context-specific AI assisstant instructions
