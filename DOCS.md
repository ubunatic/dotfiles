# Documentation Index

This document provides an index of the documentation files within the `ubunatic/dotfiles` repository, along with a brief summary of their content and purpose.

## General Documentation

*   [README.md](README.md): The main entry point for the repository, providing a general overview, usage instructions, contribution guidelines, and licensing information.
*   [CONTRIBUTING.md](CONTRIBUTING.md): Guidelines for contributing to the dotfiles repository, including coding standards, testing procedures, build system information, and the pull request process.
*   [SECURITY.md](SECURITY.md): Outlines the security policy for the repository, covering supported versions, vulnerability reporting, security considerations, and best practices.
*   [AGENTS.md](AGENTS.md): Comprehensive coding guidelines and conventions for AI agents working on the repository, including agent role, environment, repository overview, general instructions, project-specific guidelines, and detailed Bash coding guidelines.
*   [.github/copilot-instructions.md](.github/copilot-instructions.md): :warning: *Outdated - Duplicate of `AGENTS.md`* - Provides coding guidelines and conventions for AI agents (specifically Copilot) working on the repository.

## AI-Related Documentation

*   [PROMTING.md](PROMTING.md): Provides an overview of the AI prompting system in the repository, explaining `ai-prompt.sh`, `ai.mk`, how to use them, and how to define new AI skills.
*   [.ai/README.md](.ai/README.md): Explains the purpose and structure of the `.ai/` folder, which is dedicated to AI reviews, project planning, and agentic coding.
*   [.ai/current/PROJECT_202511_AgenticCoding.md](.ai/current/PROJECT_202511_AgenticCoding.md): Outlines the setup and configuration for an agentic coding environment using AI tools.
*   [.ai/instructions/README.md](.ai/instructions/README.md): States that this directory contains basic AI instructions and suggests symlinking it to `.github/instructions`.

### AI Skill Definitions

*   [.ai/instructions/simple/assist.md](.ai/instructions/simple/assist.md): Defines the "Simple Assist" skill, outlining various options for general AI assistance.
*   [.ai/instructions/simple/docs.md](.ai/instructions/simple/docs.md): Defines the "Documentation Update" skill, outlining a multi-step process for updating documentation.
*   [.ai/instructions/simple/help.md](.ai/instructions/simple/help.md): A simple instruction file for general assistance.
*   [.ai/instructions/simple/init.md](.ai/instructions/simple/init.md): Defines the "Init AI Environment" skill, for setting up and updating AI-related configurations.
*   [.ai/instructions/simple/review.md](.ai/instructions/simple/review.md): Defines the "Review AI Code Review Instructions" skill, focusing on code quality and improvements based on recent changes.
*   [.ai/instructions/simple/skills.md](.ai/instructions/simple/skills.md): Defines the "List Agentic Skills" skill, for listing and comparing AI skills.
*   [.ai/instructions/simple/tech.md](.ai/instructions/simple/tech.md): Defines the "Technology Focus" skill, for identifying and suggesting removal of non-technical content.
