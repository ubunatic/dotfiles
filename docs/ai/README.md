# AI Documentation

> **Work in progress.** Everything in this folder is experimental and
> evolving — don't take it too seriously.

This folder contains documentation, resources, and instructions for
AI-assisted work on the `ubunatic/dotfiles` repository.

## Structure

- `instructions/`: AI assistant skill definitions and context-specific
  instructions for use with Claude Code or similar CLI tools.
- `projects/`: Project plans and notes for AI-assisted work (active and
  archived).
- `prompting.md`: Overview of how to structure prompts and skills for
  AI assistants.

## Usage

The main AI instructions for this repository live in
[AGENTS.md](../../AGENTS.md) and [CLAUDE.md](../../CLAUDE.md) at the root.

Skill definitions in `instructions/simple/` can be referenced when prompting
an AI assistant to perform a specific task (e.g. code review, docs update).
